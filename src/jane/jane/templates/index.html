FDSN Web Services

Jane implements version 1.1 of all three FDSN web services.
fdsnws-station
For access to station metadata in the FDSN StationXML format
fdsnws-dataselect
For access to time series data in the MiniSEED format.
fdsnws-event
For access to event parameters in the QuakeML format.
