Jane FDSNWS DataSelect Web Service

The dataselect Web service provides access to time series data in miniSEED format. The data may be filtered e.g. by time, station and network parameters. The request type is HTTP-GET and HTTP-POST. Please refer to http://www.fdsn.org/webservice for a complete service description.

Available URLs

  • query
  • queryauth
  • version
  • application.wadl

Feature Notes

  • quality request parameter not implemented (information not available in SeisComP)
  • minimumlength request parameter not implemented
  • longestonly request parameter not implemented
  • access to restricted networks and stations is only granted through the queryauth method
